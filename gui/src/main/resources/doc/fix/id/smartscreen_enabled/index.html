Enabling SmartScreen

SmartScreen is a security feature included with Windows that monitors programs and web sites to detect when they try to access suspicious files or web servers. SmartScreen can detect and prevent attempts to install viruses or steal personal information by malicious web sites. SmartScreen is currently turned off. To turn it on, follow these steps.

  1. First, click on the start button and then the settings gear.

  2. When the Settings app opens, click on the "Search" box at the top and type in "smartscreen".

  3. Below the search box, the words "App & Browser control" will appear. Click these words.

  4. A new window will open with the title "Windows Defender Security Console".

  5. Look for the words "Check apps and files". Underneath, there should be three buttons labeled "Block", "Warn", and "Off".

  6. Click on the button labeled "Block".

  7. You may see a message that says "Do you want to allow this app to make changes to your device?". If this message appears, then press "Yes". If it does not appear, then skip this step.

  8. Close the "Windows Defender Security Console" and the Settings app. Congratulations, SmartScreen is now protecting your computer!
